# Weather and Air Quality Station

This is a comprehensive weather and air quality monitoring system built with a modular architecture using Node.js/Express for the backend and React/Vite for the frontend.

## Project Structure

```
weather-and-air-quality/
├── backend/                 # Node.js/Express backend
│   ├── server.js            # Main server file
│   └── README.md           # Backend documentation
├── frontend/                # React/Vite frontend
│   ├── src/                 # Source code
│   ├── public/              # Static assets
│   └── README.md           # Frontend documentation
├── README.md               # This file
└── package.json            # Root package.json
```

## Features

- Real-time weather monitoring
- Air quality tracking
- Data visualization with charts
- Responsive web interface
- RESTful API for data access
- ESP32 hardware integration support

## Technologies Used

### Backend
- Node.js with Express
- MongoDB for data storage
- MQTT for real-time data streaming
- CORS middleware for API access

### Frontend
- React 18 with Vite
- Chakra UI for styling
- Chart.js for data visualization
- React Router for navigation
- Material UI icons

## Getting Started

1. Clone the repository
2. Install dependencies in both backend and frontend directories
3. Set up the database
4. Configure environment variables
5. Start both the backend server and frontend development server
